# EduFinance AI

A comprehensive student finance management dashboard designed to help students evaluate education loans, track monthly expenses, plan savings goals, and make informed financial decisions through interactive analytics and visualizations.

## Overview

EduFinance AI is a modern web-based financial planning application built using HTML, CSS, and JavaScript. The platform provides a centralized dashboard where students can calculate loan EMIs, monitor expenses, analyze affordability, plan savings targets, and gain financial insights through interactive charts and reports.

The project focuses on delivering an intuitive user experience with a responsive design, glassmorphism-inspired interface, financial analytics, and data visualization capabilities.

## Core Functionalities

### EMI Formula

The application uses the standard EMI calculation formula:

```text
EMI = P × r × (1 + r)^n
      -----------------
      (1 + r)^n - 1
```

Where:

* P = Principal Loan Amount
* r = Monthly Interest Rate
* n = Number of Monthly Installments
